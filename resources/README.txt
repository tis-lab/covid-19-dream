Running make generates the following owl and tsv files:

hp-labels.tsv
    Two column HPO curie,label file, only subclasses of HP:0000118

hp-snomed.tsv
    Two column HPO curie, SNOMEDCT_US id

mondo-closures.tsv
    Two column id, superclass

mondo-cohd.tsv
    Two column Mondo curie, cohd id

mondo-labels.tsv
    Two column id, label

mondo-slim.owl
    Mondo owl with some annotations removed
